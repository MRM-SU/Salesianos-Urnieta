## Osagaien kalkukulua
$$F_{nx}=F_n \cos \theta$$
$$F_{ny}=F_n \sin \theta$$

$$tan \theta=\frac{F_{ny}}{F_{nx}} \Rightarrow \tan^-1 (\frac{F_{nx}}{F_{ny}})=\theta$$
